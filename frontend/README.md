# ClawGuard Web Frontend

静态 Bootstrap 控制台，用于演示登录、OpenClaw 任务捕获、过滤预检、人工审核、沙箱执行、结果查看、审计和策略读取。

直接用浏览器打开 `index.html`，或通过任意静态服务器托管。默认后端地址是 `http://127.0.0.1:8000`。

## 审核流

1. 登录后点击“模拟 OpenClaw 捕获”，请求只进入待审核队列，不会立即执行。
2. 系统自动完成命令过滤、输入过滤和风险判断，并给出“建议同意 / 建议修改 / 建议否决”。
3. 审核人查看队列项后，可以直接同意执行、否决，或修改参数和输入内容后重新预检。
4. 只有“同意执行”后才会生成 Job ID，并进入 Docker 沙箱运行。
