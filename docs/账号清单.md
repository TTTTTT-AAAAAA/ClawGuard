# ClawGuard 角色账号清单

> 最后更新: 2026-06-16
> 后端地址: http://127.0.0.1:8000

---

## 角色总览

系统支持 **3 种角色**，遵循三权分立的安全模型：

| 角色 | 职责 | 原则 |
|------|------|------|
| **admin** | 系统管理 | 执行任务 + 管理策略 + 查看审计 |
| **student** | 任务执行 | 提交分析任务、查看结果 |
| **auditor** | 安全审计 | 仅查看审计日志，不可执行任何操作 |

---

## 账号信息

### 1. 管理员 — admin

| 项目 | 值 |
|------|-----|
| 用户名 | `admin` |
| 密码 | `admin123` |
| 角色 | `admin` |
| 权限 | 执行任务、管理策略、查看审计、导出结果 |
| 无权限 | 禁止 shell_exec |

**用途:** 日常运维管理、策略配置、审核队列审批。

**登录:**
```powershell
curl -X POST http://127.0.0.1:8000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"admin123"}'
```

---

### 2. 普通用户 — student

| 项目 | 值 |
|------|-----|
| 用户名 | `student` |
| 密码 | `student123` |
| 角色 | `student` |
| 权限 | 执行任务 (`run_openclaw`、`analyze_file`)、查看结果 |
| 无权限 | shell_exec、export_secret、manage_policy、**view_audit** |

**用途:** 提交 OpenClaw 分析任务，查看运行结果。

**登录:**
```powershell
curl -X POST http://127.0.0.1:8000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"student","password":"student123"}'
```

---

### 3. 审计员 — auditor

| 项目 | 值 |
|------|-----|
| 用户名 | `auditor` |
| 密码 | `auditor123` |
| 角色 | `auditor` |
| 权限 | 仅 `view_audit` — 查看审计记录 |
| 无权限 | **不能执行任何任务**、不能管理策略、不能导出 |

**用途:** 安全审计员，审查系统操作记录，发现异常行为。

**登录:**
```powershell
curl -X POST http://127.0.0.1:8000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"auditor","password":"auditor123"}'
```

## 权限矩阵

| 操作 | admin | student | auditor |
|------|:-----:|:-------:|:-------:|
| `run_openclaw` — 执行任务 | ✅ | ✅ | ❌ |
| `analyze_file` — 分析文件 | ✅ | ✅ | ❌ |
| `view_result` — 查看结果 | ✅ | ✅ | ❌ |
| `export_result` — 导出结果 | ✅ | ❌ | ❌ |
| **`view_audit` — 查看审计** | ✅ | ❌ | **✅** |
| **`manage_policy` — 管理策略** | ✅ | ❌ | ❌ |
| `shell_exec` — Shell 执行 | ❌ | ❌ | ❌ |
| `export_secret` — 导出密钥 | ✅ | ❌ | ❌ |

Docker 沙箱限制:

| 限制 | admin | student | auditor |
|------|:-----:|:-------:|:-------:|
| 网络 | ✅ 有 | ❌ 无 | ❌ 无 |
| 内存 | 1024m | 512m | 256m |
| CPU | 2 | 1 | 1 |
| PIDs 限制 | 256 | 128 | 64 |
| 最大运行时间 | 300s | 60s | 30s |

## 安全提示

- ⚠️ 默认密码仅用于开发和演示环境
- ⚠️ 生产环境请修改所有默认密码
- ⚠️ `admin` 和 `auditor` 必须由不同人员持有（职责分离）
- ⚠️ 审计日志路径: `GET /api/audit?risk_level=high`
