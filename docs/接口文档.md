# 接口文档

## Auth

`POST /api/auth/login`

```json
{"username":"admin","password":"admin123"}
```

`POST /api/auth/challenge`

```json
{"uid":"student","purpose":"login"}
```

`POST /api/auth/cfl-login`

```json
{"uid":"student","nonce":"...","timestamp":1710000000,"signature":"..."}
```

`GET /api/auth/cfl/status`

返回当前 CFL 模式、DLL 路径、DLL 是否存在、DLL 架构和加载诊断。

`GET /api/auth/cfl/public-key/{uid}`

导出当前连接 UKey 的签名公钥。需要登录，管理员可查看任意 uid，普通用户只能查看自己。

## Tasks

```json
{
  "action": "run_openclaw",
  "params": {"mode": "safe", "input_name": "request.json"},
  "input_text": "{\"sample\":\"hello\"}"
}
```

`GET /api/tasks/{job_id}`

`GET /api/tasks/{job_id}/logs`

`GET /api/tasks/{job_id}/result`
