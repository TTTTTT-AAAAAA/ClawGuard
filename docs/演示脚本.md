# 演示脚本

1. 打开前端 `frontend/index.html`。
2. 使用 `admin/admin123` 登录。
3. 提交 `run_openclaw` 正常任务。
4. 查看任务状态、日志和结果。
5. 输入 `token=secret-value` 后提交，确认审计日志脱敏。
6. 输入 `-----BEGIN PRIVATE KEY-----`，确认输入过滤拒绝。
7. 参数 JSON 中加入 `{"mode":"safe","note":"curl http://x"}`，确认命令过滤拒绝。
8. 使用 `student/student123` 登录后拉取审计，确认权限拒绝。
